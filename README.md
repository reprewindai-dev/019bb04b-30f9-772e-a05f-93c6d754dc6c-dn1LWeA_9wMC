## Key Standards

### App Requirements
- ? Premium, user-friendly design
- ? Fully functional and optimized
- ? Production-ready quality
- ? Cost-friendly API integration
- ? Backend integration ready

### Duplicate Management
- Keep the most complete, feature-rich version
- Discard all other duplicates
- Regular audits to maintain quality

### Completion Rules
- **Complete every app once** - Never create same version twice
- **100% completion** required before moving to next app
- **90+ quality score** mandatory to proceed
- Minor adjustments/improvements approved
- Permission to pull from "brains and engines" for game-changing features

### iCloud Apps
- All apps must be complete and production-ready
- No incomplete or prototype apps

### Save Location
All completed apps saved to: `C:\Users\antho\Documents\[app-name]\`
